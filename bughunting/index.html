GlobaLeaks Bug Hunting

Kill bugs, get swag and 1337 credz!

Make GlobaLeaks more secure!

We invite hackers and infosecurity communities to play with the GlobaLeaks source code as a way to spot security bugs. We will be happy to reward qualifying reports by sending an Hermes Center branded USB key with Tails pre-loaded.

Security bugs we are looking for

We're looking for bugs affecting the confidentiality, integrity and availability of our users within the boundaries of the threat model. Examples of software vulnerabilities include, but are not limited to, XSS, CSRF, anonymity or privacy weaknesses, code execution, etc.

How to report bugs?

To submit security bugs, please use our HackerOne Bug Bounty page.

If you'd rather be private, you can encrypt the message with our PGP Public Key.
